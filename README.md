# 博客

## 项目搭建

- npx create-next-app （文件名）

  - 启动
    - yarn dev

## 配置支持 css 样式

    ```shell

    yarn add @zeit/next.css
    ```

### 配置文件

- .babelrc
- .next.config.js
  - 两个文件就是为了适应 css 和 ant 第三方包

## 安装有关第三方包

    ```shell
        **Css样式**
        yarn add antd
         * 安装如果不是一个控制台的时候引入需要暂停重启
        **按预加载**
        yarn add babel-plugin-import
    ```
